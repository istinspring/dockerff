* Free software: BSD license
* Documentation: https://dockerff.readthedocs.io.

Pre-requirements
----------------

+ Docker 1.12+
+ Docker-compose 1.8+


Features
--------

* [ ] Fluentd to aggregate logs from the containers
* [ ] Start/Stop required docker instances
* [ ] Plugable scenarios for selenium
* [ ] Something else
